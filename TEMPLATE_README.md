# Mews — Open-source Curator Template

**Mews** is a clean, reusable template for building your own curator site with bookmarks, portfolio, and public/private research shelves.

Built by Taron & Sunshine for creators, researchers, and industry specialists.

## What You Get

- **Curated shelves** — organize bookmarks by topic
- **Portfolio view** — showcase your work
- **Public + private** — share what matters, keep research local
- **Magpie integration** — sync X bookmarks automatically
- **Deploy instantly** — to GitHub Pages or Vercel

## Quick Start

1. **Fork this repo** and rename to `[your]_Mews`
2. **Read SETUP.md** for customization steps
3. **Deploy** to GitHub Pages (free, one-click)

See SETUP.md for full instructions.

## Features

✅ Responsive design  
✅ Dark theme by default  
✅ Magpie bookmark integration  
✅ GitHub Pages ready  
✅ Zero dependencies (pure HTML/CSS/JS)  

## Customize

- `index.html` — your name, bio, links
- `shared-shelf/` — add bookmark collections
- `books/` — reading notes and highlights
- `portfolio/` — your work/projects

## Deploy

**GitHub Pages (free):**
1. Push to `main`
2. Go to repo Settings → Pages
3. Enable GitHub Pages for `main` branch
4. Done — site goes live in ~1 minute

**Vercel (free):**
1. Connect repo to Vercel
2. Auto-deploys on every push
